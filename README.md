写在前面
----
此API接口可以返回与搜索关键词有关的影视数据、音频数据和文学数据。此接口涉及到的软件与技术包括 MySQL、PHP、Redis、Nginx.
API 地址
------
```API
https://www.6zgm.com/api.html
```

调用方法
----

同时支持 GET 和 POST 方法调用。

传入参数
----
'key'：搜索词；

返回数据类型（数据结构）
------
Json (Key-Value 键值对)

返回数据分析
------

 1. HD：返回标清影视数据（Array类型）

 3. song: 返回歌曲数据 （Array类型）
![12212-izcnyxnk05.png](https://www.6zgm.com/usr/uploads/2020/04/1466987354.png)
 4. poet：返回古诗词数据 （Array类型）
![35341-7lwecu1smwa.png](https://www.6zgm.com/usr/uploads/2020/04/61633410.png)

![01888-b3gmabriza.png](https://www.6zgm.com/usr/uploads/2020/04/3594647068.png)
